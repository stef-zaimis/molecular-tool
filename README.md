# Molecular Diagnosis Tool

A small Python desktop tool for identifying molecular diagnostic characters from aligned FASTA files.

## Features

- Reads aligned FASTA files
- Identifies focal-set fixed sites
- Filters globally conserved sites
- Reports 1-site and 2-site diagnostic characters
- Searches 5-site combinations for similarity optimisation
- Exports a text report
- Exports a formatted Excel workbook
- Includes a Tkinter GUI and FASTA viewer
